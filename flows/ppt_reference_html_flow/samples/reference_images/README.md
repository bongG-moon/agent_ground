이 폴더의 PNG 세 장은 `ppt_reference_html_flow`의 첫 실행과 디자인 분석 검증을 위한 합성 샘플입니다. 실제 회사 자료나 실제 실적 데이터가 아닙니다.

| 파일 | 역할 | 업로드 위치 |
| --- | --- | --- |
| `reference_cover_navy_teal.png` | 네이비·틸·앰버 표지 디자인 | 표지 Multi Image Base64 Encoder |
| `reference_body_trend.png` | 단일 추세 차트 중심 본문 | 본문 Multi Image Base64 Encoder 첫 번째 |

공통 생성 조건:

- 16:9 landscape, 1672×941 PNG
- 로고, 회사명, 읽을 수 있는 문장과 실제 숫자 없음
- gradient, emoji, watermark와 장식 과다 배제
- 표지에는 넓은 여백과 강한 제목 영역, 본문에는 하나의 주 시각 구조 사용

이미지는 OpenAI 내장 이미지 생성 도구로 만든 프로젝트용 샘플입니다. 결과물의 디자인 배치만 참고하며 이미지 안 도형은 발표의 실제 데이터로 해석하지 않습니다.
