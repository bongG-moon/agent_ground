EWS·DRM Outlook 메일 요약 Flow

사내 EWS와 NTLM 인증으로 Outlook 메일 본문·파일 첨부를 읽고, 첨부를 허용된 DRM text API의 평문 TXT로 변환한 뒤 메일별 분석과 전체 업무 요약을 생성합니다.

Flow JSON 열기사용 가이드 열기
현재 상태는 user_testing입니다.정적 schema와 격리 계약 검증 범위는 manifest에 기록되어 있으며, 실제 사내 endpoint·자격증명·업무 파일을 사용한 확인과 사용자 완료 승인은 별도입니다.

Configuration

첫 실행 전에 입력할 값

  • Configure EWS URL, mailbox, AD account, password, and the internal Nexus settings used by the actual environment.
  • Prefer enabling TLS verification with the internal CA bundle after confirming the current EWS certificate chain.
  • Configure the approved DRM text API URL, Bearer token, employee number, and exact allowed host.
  • For an HTTP-only closed test network, explicitly enable insecure HTTP after security approval.
  • Confirm the DRM API text extraction coverage for PDF, Office, HWP, text, CSV, and image attachments.
  • Select the same approved internal or local model in both Language Model nodes.
