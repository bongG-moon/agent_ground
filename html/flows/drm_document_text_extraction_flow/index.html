DRM 문서 텍스트 추출 Flow

문서·텍스트·일반 이미지 파일을 업로드하면 허용된 사내 DRM text API에 전달하고 반환된 평문을 LLM 없이 출력합니다.

Flow JSON 열기사용 가이드 열기
현재 상태는 user_testing입니다.정적 schema와 격리 계약 검증 범위는 manifest에 기록되어 있으며, 실제 사내 endpoint·자격증명·업무 파일을 사용한 확인과 사용자 완료 승인은 별도입니다.

Configuration

첫 실행 전에 입력할 값

  • Enter the approved DRM text API URL, Bearer token, employee number, and an exact allowed host.
  • Keep HTTPS and TLS verification enabled unless the approved closed test network only supports HTTP.
  • Confirm that the DRM API returns plain text for each enabled document or image format, including OCR text for images when required.
